How to Create a Table

Tag Name Description
<table> table wrapper element for all HTML tables
<thead> table head set of rows defining the column headers in a table
<tbody> table body set of rowss containing acctual table data
<tr> table row table row container (row across table)
<td> table data what you want inside the row
<foot> table foot set of rows defining the footer in a table

Table Attributes

Attribute Name Description
colspan column span defines how many columns a td element should
rowspan row span defines how many rows a td element should span

Structure Elements

Syntax Name Description
<header> header holds Nav links and intro content
<nav> nav links can use inside header or on it's own
<main main main content in HTML page
<footer> footer ending content/info found at the bottom of a page. can hold a "back to top" button
<section> section an area that defines element sections (chapters, headings, etc.)
<article> article the content within the section element (paragraphs in the section)
<aside> aside additional info/enhances elements (endnotes, comments, pull-quotes)
<figure> figure adds media! *use <img src="">;
<audio> audio adds sound! *use <source src="">;
<embed> embed self-closing; can be added to other websites
<a> anchor the text itself is a link; should be descriptive of link source

Structure Attributes

Name Description
title adds descriptive text to a picture when hovered over
class allows specific CSS editing to a html elements
id more specific than class; affects the entire html structure
